# راهنمای کامل اتصال به اینترنت بین‌الملل با استفاده از AmneziaWG

این راهنما به شما کمک می‌کند تا با استفاده از برنامه AmneziaWG و WireSock، به سادگی به اینترنت بین‌الملل متصل شوید.

## پیش‌نیازها

قبل از شروع، لطفاً برنامه‌های مورد نیاز را بر اساس سیستم عامل خود دانلود و نصب کنید:

*   **اندروید:**
    *   دانلود برنامه AmneziaWG از Google Play: [دانلود از Google Play](https://play.google.com/store/apps/details?id=org.amnezia.awg&hl=fa)
*   **آیفون:**
    *   دانلود برنامه AmneziaWG از App Store: [دانلود از App Store](https://apps.apple.com/us/app/amneziawg/id6478942365)
*   **ویندوز:**
    *   دانلود برنامه WireSock: [دانلود برنامه WireSock](https://www.wiresock.net/wiresock-secure-connect/download)

## مراحل اتصال

### ۱. دریافت کانفیگ اولیه

برای دریافت فایل کانفیگ اولیه، مراحل زیر را دنبال کنید:

*   به وب‌سایت [Warp Generator](https://warp-generator-config.vercel.app/) مراجعه کنید.
*   روی گزینه **AWG 2.0 v3** کلیک کنید تا فایل کانفیگ شما تولید شود.

### ۲. دریافت EndPoint مناسب

برای اطمینان از اتصال پایدار، لازم است یک EndPoint مناسب را انتخاب کنید:

*   به وب‌سایت [سایت EndPointها](https://ircfspace.github.io/endpoint/) مراجعه کنید.
*   یک EndPoint از نوع **Warp IPv4** انتخاب کنید. این مورد برای اتصال شما ضروری است.

### ۳. تنظیم و اتصال

حالا نوبت به تنظیمات نهایی و برقراری اتصال است:

#### برای اندروید و آیفون:

1.  فایل کانفیگ دانلود شده را در برنامه AmneziaWG ایمپورت (Import) کنید.
2.  وارد بخش ویرایش (Edit) کانفیگ شوید.
3.  مقدار **EndPoint** را با لینکی که در مرحله قبل دریافت کردید، جایگزین کنید.
4.  اتصال را با زدن دکمه Start برقرار کنید.

#### برای ویندوز (با استفاده از WireSock):

1.  برنامه WireSock را نصب کنید.
2.  فایل کانفیگ WireSock را وارد برنامه WireSock کنید.
3.  به بخش تنظیمات (Settings) برنامه بروید.
4.  گزینه **Edit Text** را انتخاب کنید.
5.  به انتهای فایل متنی اسکرول کنید و مقدار **EndPoint** را با لینکی که از سایت EndPointها دریافت کردید، جایگزین کنید.
6.  تغییرات را ذخیره (Save) کنید و سپس اتصال را برقرار نمایید.

**نکته مهم:** حتماً از EndPoint نوع **Warp IPv4** استفاده کنید تا اتصال شما به درستی انجام شود.
